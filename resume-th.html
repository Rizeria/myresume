บริษัท อีกาซีน จำกัด

พัฒนาเว็บไซต์ตามความต้องการของลูกค้าและ ดูแลเว็บไซต์ที่บริษัทพัฒนา(MA)

  • สร้างเว็บแอปพลิเคชั่นด้วย JQuery + PHP
  • ออกแบบและสร้างฐานข้อมูล MYSQL และ MSSQL.
  • สร้างหน้าเว็บตาม Wireframe Design โดยใช้ Bootstrap
  • สร้าง Web Service ด้วย PHP, jQuery Ajax
  • สร้าง RESTfulAPI ด้วย PHP(OOP) กับฐานข้อมูล MySQL
  • สร้างหน้าแสดงไฟล์วัตถุ 3 มิติ ด้วย three.js (Javascript 3D library)
  • ใช้ (mod_rewrite) htaccess file ของ Apache และ IIS
  • สร้างหน้าแบบสำรวจข้อมูล บันทึกข้อมูลลงฐานข้อมูล SQLite
  • สร้างฟอร์มนำเข้าข้อมูลจากไฟล์ Excel บันทึกลงฐานข้อมูล MySQL
  • คิวรี่ข้อมูลจากฐานข้อมูล ออกเป็นรายงานในรูปแบบไฟล์ Excel,PDF
